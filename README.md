# 🏝️ マニラTrip：7日間完全ガイド

フィリピン・マニラの7日間旅行プランを視覚的かつ共有しやすい形で表示する**モバイル最適化**Webサイトです。

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://kizuna42.github.io/manila-trip-plan/)
[![Mobile Optimized](https://img.shields.io/badge/Mobile-Optimized-blue)](#)
[![Responsive Design](https://img.shields.io/badge/Responsive-Design-orange)](#)

## ✨ 特徴

### 🎨 デザイン
- **南国テーマ**: トロピカルな配色とハワイアンスタイルのヘッダー
- **完全レスポンシブ**: モバイル・タブレット・デスクトップ対応
- **モバイル最適化**: 480px以下・360px以下の画面サイズに特別対応
- **インタラクティブUI**: ハンバーガーメニュー・展開式タイムライン

### 📱 モバイル体験
- **テキスト改行最適化**: 自然な改行制御で読みやすさ向上
- **コンパクトレイアウト**: 密度の高い情報表示
- **タッチ最適化**: スマホ操作に配慮したインターフェース
- **カード式表示**: テーブルをモバイル向けカードに自動変換

### 🗓️ 機能
- **インタラクティブ日程表**: 各日の詳細を展開/折りたたみ
- **予算自動計算**: 1人あたりの詳細な費用見積もり
- **参考資料リンク**: 現地情報への便利なリンク集
- **選択式プラン**: 日帰り旅行の複数選択肢

## 📅 旅行概要

- **📆 期間**: 2025年8月18日〜24日（7日間）
- **👥 参加者**: 大学生3人
- **🌍 行き先**: フィリピン・マニラ
- **💰 予算**: 約53,300円/人（20,500 PHP）

## 🚀 デプロイ & 公開

✅ **現在公開中**: https://kizuna42.github.io/manila-trip-plan/

#### セットアップ手順:
1. **リポジトリ作成**: GitHubで新しいリポジトリを作成
2. **ファイルアップロード**: プロジェクトファイルをpush
3. **Pages設定**: Settings > Pages > Source「Deploy from a branch」
4. **ブランチ選択**: 「main」ブランチを選択
5. **自動デプロイ**: 数分後にサイトが公開される

## 💡 使い方

### 🖥️ デスクトップ
- **ナビゲーション**: 上部メニューから各セクションに移動
- **日程詳細**: 各日のカードをクリックして詳細を展開
- **スムーズスクロール**: リンククリックで自動スクロール
- **ホバーエフェクト**: カードやボタンにマウスを乗せると反応

### 📱 モバイル
- **ハンバーガーメニュー**: 右上の≡ボタンでナビゲーション展開
- **タッチ操作**: タップで日程カード展開、スワイプでスクロール
- **カード表示**: テーブル情報が見やすいカード形式に自動変換
- **テキスト最適化**: 改行制御により読みやすい表示

### ⌨️ キーボードショートカット
- **Esc**: 展開されたコンテンツを閉じる / メニューを閉じる
- **Space**: 次のセクションにスクロール
- **Tab**: フォーカス移動（アクセシビリティ対応）

## 🎨 カスタマイズ

### 🌈 カラーテーマ変更

```css

    /* 南国東南アジアカラーパレット */

}
```
